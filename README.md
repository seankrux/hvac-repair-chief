# HVAC Repair Chief

Professional HVAC website for HVAC Repair Chief serving Boca Raton and Fort Lauderdale, FL.

## Tech Stack
- Next.js 16, React 19, TypeScript
- Tailwind CSS 4, shadcn/ui
- Vercel Analytics + Speed Insights

## Live Site
https://hvac-repair-chief.vercel.app

## Core Scripts

```bash
npm run dev         # local development
npm run lint        # ESLint
npm run typecheck   # TypeScript validation
npm run build       # production build
```

## Local Development

```bash
npm ci
npm run dev
```

## Repo Structure

- `src/app` — route files for homepage, services, and locations
- `src/components` — reusable marketing site components
- `src/config/brand.ts` — shared brand identity and theme
- `src/config/services.ts` — service definitions
- `src/config/locations/` — per-city location data
